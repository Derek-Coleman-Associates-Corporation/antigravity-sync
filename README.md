This repository contains the physical persistent state for the Antigravity local agent.

1. Open terminal on your new Mac.
2. Clone this repository anywhere.
3. Physically copy the contents of this repository directly into the target Antigravity system folder:
   `cp -R * ~/.gemini/antigravity/brain/`
4. The moment you launch Antigravity on the new Mac, it will natively ingest every historical conversation and topological memory file stored in these UUID directories!
